# Layers EDX

## Description
This project is a Python port of parts of the [NIST EPQ (Electron Probe Quantification) library](https://github.com/usnistgov/EPQ). It provides tools for X-ray microanalysis and quantification.

## Structure
- `layers_edx/`: The core Python package containing the ported EPQ functionality.
- `src/`: Scripts and usage examples (e.g., `main.py` for running simulations/quantification).
- `test/`: Unit tests.
